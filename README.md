# GPU Market Tracker

Real-time GPU price tracker for the Chilean market. Built with Java.

## Tech Stack:
* **Backend:** Java, Spring Boot, Spring WebFlux (WebClient)
* **Database:** PostgreSQL + Spring Data JPA
* **Frontend:** Angular.
* **Data Ingestion:** REST API Integration & Web Scraping (SoloTodo, etc.)

## Key Features
* **Automated Tracking:** Scheduled data ingestion from major Chilean retailers and international markets to keep prices up to date.
* **Historical Data Analytics:** Time-series visualization to track price drops, identify trends, and find the best time to buy.
* **Smart Recommendations:** AI-powered assistant to provide personalized purchase advice based on real-time market data.
