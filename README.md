# GPU Market Tracker

Real-time GPU price tracker for Chilean and international markets. Built with Java.

## Tech Stack:
- **Backend:** Java Spring boot + PostgreSQL
- **Frontend:** Angular.

## Key Features:
- Automated tracking for Chilean retailers and International market.
- Time-series visualization and historical price tracking.
- AI Assistan for smart purchase recommendations based on real-time data.
